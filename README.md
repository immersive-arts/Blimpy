# blimpy

## Connectors

All:

- Type: JST PH RA (S2B-PH-SM4)
- Pitch: 2.00mm (0.079in)
- Current: 2 A
- Wire Size: 32 to 24 AWG

## Cables

Actors:

- Wire Gauge: 26
- Cross Section: 0.129mm2
- Max Current: 2.2A

Battery:

- Wire Gauge: 24
- Cross Section: 0.205mm2
- Max Current: 3.5A
